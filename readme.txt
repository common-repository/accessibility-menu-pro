Improve your menu html markup to be more accessible and compliant. Customize colors, fonts, and more to match your site's unique style.

== Description ==

Accessibility Menu Pro replaces the standard menus in your WordPress theme with an American Disabilities Association (ADA) AA-compliant menu. All HTML semantics are compliant with accessibility standards, ensuring that your site is more accessible to users with disabilities.

= Features =
* Replaces default theme menus with ADA AA-compliant menus.
* Ensures all HTML semantics meet accessibility standards.
* Easy integration with existing WordPress themes.
* Fully customizable to match the look and feel of your site.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/accessibility-menu-pro` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. Use the Dashboard -> Settings -> Accessibility Menu Pro screen to configure the plugin.

== Frequently Asked Questions ==

= How do I customize the menu? =

You can customize the menu using the settings available in the Settings -> Accessibility Menu Pro screen. For advanced customization, you can modify the CSS styles in your theme or child theme.

= Does this plugin work with all themes? =

Accessibility Menu Pro is designed to work with most WordPress themes. However, some themes with heavily customized menu structures might require additional adjustments.
